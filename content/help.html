Basic Usage

Copy challenges from Apex, include star value for each.
Hotkey N will add a new challenge, pressing Enter while editing will save the challenge.
When adding a new challenge, the first number entered will be placed in the max value box, if no other number has been entered already.
The search icon opens a filter panel, which will search all entered challenges. Text after a comma will be filtered as Modes. Ex: "Bloodhound, BR" shows all challenges with Bloodhound and the BR mode.

Optimal Path

This shows you what legends, weapons, and modes you should play to maximize your point gain.
Clicking on any number or title will apply that filter to the challenge list so you can view the applicable challenges.
